strona 13/51
3.2 Google API
Google API to zestaw interfejsów, zaprojektowanych przez firme Google, umozliwiajacy
dostep i integracje z wieloma oferowanymi przez nich uslugami oraz narzedziami, takimi
jak wyszukiwarka czy mapy.
3.2.1
Google Maps
Google Maps API to nowoczesna usluga, która od 2005 roku pozwala na osadzenie map
w aplikacjach tworzonych na urzadzenia mobilne[8]. Programisci Google udostepnili
przykladowe kody zródlowe, prezentujac wszystkie mozliwosci interfejsu, dzieki czemu
dostosowanie mapy do wlasnych potrzeb jest stosunkowo proste, dajac jednoczesnie
mozliwosc manipulacji wieloma jej ustawieniami, np. ustawieniem kamery czy
sposobem prezentacji samej trasy.
3.2.2
Google Maps Direction
Google Maps Directions API to usluga wyznaczajaca trase pomiedzy podanymi
lokacjami oparta o zadanie HTTP. Dostepne jest kilka trybów oraz mozliwosc
parametryzacji wyszukiwania trasy. W odpowiedzi odeslane zostaja, w formacie JSON
lub XML, informacje odnosnie trasy przejazdu, dlugosci tej trasy oraz szacowanego
czasu, jaki zajmie dotarcie do celu.
